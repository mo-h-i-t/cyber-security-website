How Cybersecurity Works

Cyber security works by protecting computers, networks, applications, and data from unauthorized access, attacks, or damage. It combines technology, processes, and human awareness to defend against cyber threats.

Layers of Cyber Security

↬ Network Security → Protects internal networks
↬ Application Security → Secure software design
↬ Endpoint Security → Protects devices (laptops, phones)
↬ Data Security → Encrypts sensitive data
↬ Identity & Access → Confirms authorized access
↬ Cloud Security → Protects cloud resources
↬ Human Factor → Anti-phishing training

Protection Mechanisms

↬ Prevention → Firewalls, antivirus, strong auth
↬ Detection → Behavioral monitoring tools
↬ Response → Incident management & triage
↬ Recovery → Backup restoration & recovery

Types of Threats

↬ Malware (viruses, ransomware)
↬ Phishing & Social Engineering
↬ DDoS Attacks (overloading servers)
↬ Man-in-the-Middle Attacks
↬ Zero-day Exploits
↬ Insider Threats

Security Best Practices

↬ Regular software updates & patches
↬ Data encryption (at rest & transit)
↬ Backup & disaster recovery plans
↬ Monitoring & early detection logs
↬ Security awareness user training
